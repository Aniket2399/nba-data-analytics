# 🚀 Quick Start Guide

## ⏱️ 5-Minute Setup

### **Step 1: Prepare Data**
```bash
# Download from: https://www.kaggle.com/datasets/wyattowalsh/basketball
# Save the file as: data/basketball.sqlite
```

### **Step 2: Install Dependencies**
```bash
pip install -r requirements.txt
cd frontend
npm install
cd ..
```

### **Step 3: Run Pipeline**
```bash
python ingestion/ingest.py
cd transform && dbt build --profiles-dir . && cd ..
python tableau/export_marts.py
```

### **Step 4: Start Dashboard**
```bash
cd frontend
npm run dev
```

**Open:** http://localhost:3000

---

## 📊 Testing with Sample Data

If you don't have Kaggle data yet, the dashboard works with sample data:

```bash
# The exports/ folder contains sample CSV files
cd frontend
npm run dev
# Dashboard loads sample data automatically
```

---

## 🔄 Once Everything is Running

1. **Add your Kaggle data** → Place `basketball.sqlite` in `data/` folder
2. **Re-run pipeline** → `python ingestion/ingest.py && cd transform && dbt build && cd ..`
3. **Export data** → `python tableau/export_marts.py`
4. **Dashboard updates** with your data!

---

## 🎯 Key Files

- `data/basketball.sqlite` ← Your Kaggle data goes here
- `nba.duckdb` ← Created after running ingest.py
- `exports/*.csv` ← Generated by export_marts.py
- `frontend/src/App.tsx` ← Dashboard code

---

## ⚠️ Common Issues

| Issue | Solution |
|---|---|
| "basketball.sqlite not found" | Download from Kaggle and place in `data/` |
| "Port 3000 already in use" | Edit `frontend/vite.config.ts` to change port |
| "dbt build fails" | Make sure `python ingestion/ingest.py` succeeded |
| "No data in dashboard" | Run `python tableau/export_marts.py` |
