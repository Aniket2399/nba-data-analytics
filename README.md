To keep the download small, the large source-data files were **excluded** from this archive. The React dashboard still runs out-of-the-box because the exported CSVs in `frontend/public/data/` are bundled. You only need the files below if you want to **re-run the data pipeline**.

| Excluded file | Size | Where to put it | How to get it |
|---|---|---|---|
| `data/basketball.sqlite` | ~2.2 GB | `data/basketball.sqlite` | [Kaggle: wyattowalsh/basketball](https://www.kaggle.com/datasets/wyattowalsh/basketball) |
| `nba.duckdb` | ~706 MB | project root (`./nba.duckdb`) | Auto-generated by the pipeline (step below) |
| `frontend/node_modules` | ~82 MB | `frontend/node_modules` | Auto-generated by `npm install` |

### View the dashboard immediately (no data download needed)
```bash
cd frontend
npm install
npm run dev
```

### Re-create the data + `nba.duckdb` from scratch
```bash
# 1. Download basketball.sqlite from Kaggle and place it at data/basketball.sqlite
mkdir -p data        # then move the downloaded file into ./data/

# 2. Build the warehouse + CSVs (this generates ./nba.duckdb)
pip install -r requirements.txt
python ingestion/ingest.py
cd transform && dbt build --profiles-dir . && cd ..
python tableau/export_marts.py     # writes CSVs to exports/ and frontend/public/data/
```

## 🚀 Quick Start (5 Minutes)

### **1. Download Kaggle Data**
```bash
# Download from: https://www.kaggle.com/datasets/wyattowalsh/basketball
# Save as: data/basketball.sqlite
```

### **2. Run Data Pipeline**
```bash
# Install Python dependencies
pip install -r requirements.txt

# Run ingestion + transformation
python ingestion/ingest.py
cd transform && dbt build --profiles-dir . && cd ..

# Export to CSV for dashboard
python tableau/export_marts.py
```

### **3. Start Frontend**
```bash
cd frontend
npm install
npm run dev
# Opens http://localhost:3000
```

**Done!** Your dashboard is running with your NBA data.

## ❓ Troubleshooting

**"basketball.sqlite not found"**
→ Download from Kaggle and place in `data/` folder

**"Port 3000 already in use"**
→ Change port in `frontend/vite.config.ts`

**"dbt build fails"**
→ Make sure `python ingestion/ingest.py` completed successfully

**"CSVs not loading in dashboard"**
→ Run `python tableau/export_marts.py` first
